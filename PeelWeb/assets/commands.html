OrangePeel commands

Bot Related

>uptime
This command shows the bot's uptime.

>ping
Find out how slow the bots running

>stats
Take a look at the statistics!

>report [bug]
report a bug with the bot

>suggest [idea]
Suggest a feature for the bot

>invite
Invite me!!

Moderation and Administration

>mute [@user]
Mute a user, requires setup first:
>setupMute

>unmute [@user]
Unmute a user, requires setup first:
>setupMute

>kick [@user]
kick someone of the server

>ban [@user]
ban someone of the server

>unban [@user]
unban someone of the server

>setup
sets the server up for you

>setupMute
sets the server up for you

>autorole
sets up autorole

>nickall [name]
change the nickname of all the users to a specified string

>cmd
makes a command that will return a specified string, e.g.
>cmd greet Hello there!
makes a command called >greet
